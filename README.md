# HR Employee Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This interactive Power BI dashboard provides a deep-dive analysis of HR employee data. It is designed to help HR managers and stakeholders track key workforce metrics, understand employee demographics, analyze income distributions, and optimize talent retention strategies.

## 📷 Dashboard Preview
![HR Dashboard](dashboard_screenshot.png)

## 📊 Key Insights & Features
* **Key Performance Indicators (KPIs):** Displays Total Employee Count (1.47K), Total Monthly Income ($10M), Average Years at Company, and Primary Salary Slabs.
* **Demographics Analysis:** Utilizes a Donut Chart to show employee distribution across different age groups (highest concentration in the 26-35 range).
* **Marital Status & Job Roles:** A Stacked Column Chart analyzing how job roles and marital status vary across Departments (R&D, Sales, HR).
* **Income Distribution:** Tracks employee counts against monthly income ranges to detect pay equity and trends.

## 🛠️ Tech Stack Used
* **Tool:** Power BI Desktop
* **Data Sources:** HR Employee CSV/Excel Dataset
* **Techniques:** Data Cleaning (Power Query), Data Modeling, DAX Measures, and Interactive UI/UX Design.

## 🚀 How to View the Project
1. Download the `HR-Employee-Data-Analysis-Dashboard.pbix` file from this repository.
